# Jeu de mémoire

Petit jeu de mémoire en HTML, CSS et JavaScript vanilla.

## Fonctionnalités
- 3 niveaux de difficulté : facile, moyen, difficile
- Compteur de coups
- Compteur de paires trouvées
- Chronomètre de partie
- Bouton pour relancer une nouvelle partie

## Lancer le projet
1. Ouvrir le fichier `index.html` dans un navigateur.
2. Choisir une difficulté puis jouer.
